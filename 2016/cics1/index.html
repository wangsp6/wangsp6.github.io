CICS中的服务端、客户端、网关与应用

概述

  1. cics server
    cics服务端。负责接收并处理客户端请求

  2. ctg (cics Transaction gateway)
    cics交易网关。由client daemon和gateway daemon组成

  3. client daemon
    客户端守护进程。负责与cics服务端通讯

  4. gateway daemon
    网关守护进程。监听远程客户端调用，并通过client daemon调用cics服务端

  5. client app
    客户端应用程序。通过api调用gateway daemon或者client daemon

客户端调用方式

  1. 对于c lang，有2种访问方式
    • 通过客户端守护进程调用。需要CTG与应用客户端在同一台主机
    • 通过网关守护进程调用
  2. 对于java，只能通过网关守护进程访问
