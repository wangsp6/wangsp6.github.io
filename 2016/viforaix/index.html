vi for aix

基本命令

重复
大多数指令可以在输入前增加一个数字来扩大影响范围。
例如5yy表示复制5行,3dw表示删除3个单词

文件命令

命令 功能 说明 推荐
vi file 打开文件file 基本
view file 只读模式打开文件file 不能用:e重新打开 基本
:q 退出 文件修改后无法使用 低频
:q! 不保存退出 基本
:w 保存 基本
:w file 另存为文件file 重要文件修改前应备份 低频
:wq 保存并退出 基本
:m,n w file 将m-n行另存为文件file 低频
:m,n w >> file 将m-n行追加至文件结尾 低频
:e 重新打开当前文件 view模式下无效。常用于刷新日志 基本
:e file 关闭当前文件并打开文件file 低频
:e! 取消所有修改重新打开文件 低频
Ctrl+g 显示当前文件信息 查看文件名 低频

大词/小词
大词是以空白字符（空格/Tab）分隔的字符串，比如”AA-BB CC-DD”是两个词;
而对小词来说则是四个词。

高级搜索与替换

s命令(语法类似sed命令)

  • 语法 :m,ns/str1/str2/gc
  • 功能 在m-n行将str1替换为str2
  • 说明
    g代表每一行替换多次(可选)
    c代表需要确认(可选)
    /分隔符可替换为其他字符(通常用于替换或被替换的字符中包含/的情况)
  • 举例
    1,$s/^/#/g 替换全部行首为#(注释脚本全文)
    1,$s,^,//,g 注释全部代码

global命令

  • 语法 :m,ng/str1/d
  • 功能 在m-n行内删除包含str1的整行

##　应用举例

单条命令

命令 功能 说明
ddp 交换2行 dd+p
1,$ s/\s\+$//g 删除行尾空格 \s+$ 表示连续多个空白字符至行末
1,$ s/^M//g 删除全部windows换行符 ^M输入方式为Ctrl+v Ctrl+m

复合命令

  • 逐词替换
步骤 命令 说明
step1 /str1 搜索单词str1
step2 cw str2 将str1替换为str2
step3 n 继续查找下一个str1位置
step4 . 重复上一次替换操作
step5 重复step3-step4
  • 跟踪日志
步骤 命令 说明
step1 vi xxx.log 打开日志文件
step2 :$ 跳转至文件结尾
step3 ?str1 搜索str1
step4 :e 重新打开文件(日志文件写入新内容后)
step5 :$ 跳转至文件结尾
step6 n 重复上一次搜索str1
step7 重复step4-step6
